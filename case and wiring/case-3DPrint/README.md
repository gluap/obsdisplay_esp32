# 3D Printable case

This case is based on ["Bifr√∂st" by NoaNoa](https://www.printables.com/model/434061-bifrost-led-matrix-case/files). Observe [the CC-By-NC license](case-3DPrint/LICENSE.md).

You need to print the following files:

- [case_split_usb_c_right.stl](case_split_usb_c_right.stl) the right half of the case including a mount for the ESP32 (30-pin-version) and a hole for a standalone USB-C-plug.
- [case-3DPrint/bifrost-led-matrix-case-model_files/case_split_left.stl](case-3DPrint/bifrost-led-matrix-case-model_files/case_split_left.stl) the left part of the case without any holes.
- and either:
    - [case-3DPrint/bifrost-led-matrix-case-model_files/diffuser_split_left.stl](case-3DPrint/bifrost-led-matrix-case-model_files/diffuser_split_left.stl) The left part of the diffuser.
    - [case-3DPrint/bifrost-led-matrix-case-model_files/diffuser_split_right.stl](case-3DPrint/bifrost-led-matrix-case-model_files/diffuser_split_left.stl) The right part of the diffuser.
- or if you have a large enough print bed:
    - [case-3DPrint/bifrost-led-matrix-case-model_files/diffuser.stl](case-3DPrint/bifrost-led-matrix-case-model_files/diffuser.stl) the diffuser as a single piece.

The case can be glued with superglue.
