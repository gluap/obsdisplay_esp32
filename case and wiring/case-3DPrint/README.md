You need to print the following files:

## Case
- [case_split_usb_c_right.stl](case_split_usb_c_right.stl) the right half of the case including a mount for the ESP32 (30-pin-version) and a hole for a standalone USB-C-plug.
- [case-3DPrint/bifrost-led-matrix-case-model_files/case_split_left.stl](case-3DPrint/bifrost-led-matrix-case-model_files/case_split_left.stl) the left part of the case without any holes.

## Diffuser
The diffuser can be printed out of white PETG. If possible, changing to black filament once the diffusing layer is printed will greatly reduce color bleed between pixels. A well-calibrated first layer is required. For the diffuser print:

either:

  - [case-3DPrint/bifrost-led-matrix-case-model_files/diffuser_split_left.stl](case-3DPrint/frost-led-matrix-case-model_files/diffuser_split_left.stl) The left part of the diffuser.
  - [case-3DPrint/bifrost-led-matrix-case-model_files/diffuser_split_right.stl](case-3DPrint/frost-led-matrix-case-model_files/diffuser_split_left.stl) The right part of the diffuser.

or if you have a large enough print bed:
 - [case-3DPrint/bifrost-led-matrix-case-model_files/diffuser.stl](case-3DPrint/bifrost-led-matrix-case-model_files/diffuser.stl) the diffuser as a single piece.

Glue the case along the middle with superglue.
