# Hardware build manual

## Required parts:

- ESP32 module, 30-Pin version.
- 3 cherry type keyswitches
- 2 m3x6 screws
- [USB-C Module](https://www.amazon.de/dp/B0F2MNT26V)
- some wire
- about 10 ST-XH Crimps (BXH-001T-P0.6)
- 3-Pin JST-XH Board connector [male]
- 3-Pin JST-XH connector [female]
- 5-Pin JST-XH Board connector [male]
- 5-Pin JST-XH connector [female]
- for the 3D-printed case version:
    - superglue
    - [the 3D-printed parts](case-3DPrint)
- For the partly-lasercut case Version:
    - wood glue
    - 8x m3x12mm bolts
    - 8x m3 nuts
    - 4x m3x8mm bolts
    - [the lasercut parts](case-Lasercut)



## Required Tools
- JST-XH Crimping pliers.
- soldering iron.

## Case

There are two case options:

- [Fully 3D-printable version.](case-3DPrint)
- [Partly 3D-printed and partly laser-cut version.](case-Lasercut)
